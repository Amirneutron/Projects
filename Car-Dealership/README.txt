The project consists of an implementation of a system for a Car dealership

The main class  will store cars in an array, that will then be used to:

 1. Register a car.
 2. Print all cars.
 3. Print a car's information.
 4. Print the cheapest car's information.
 5. Test-drive a car
 6. Fill a car's gas tank
 7. Hire a mechanic
 8. Repair a car
 9. Quit this program.


Each mechanic has a name and a type. A mechanic type depends on the type of cars he/she can fix. The three types of mechanics are:
LC mechanic (fixes light cars)
MC mechanic (fixes medium cars)
HC mechanic (fixes heavy cars)
